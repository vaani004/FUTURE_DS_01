# Business Sales Performance Analytics

## Project Overview
This project analyzes business sales data using Python and Power BI to identify sales trends, profit performance, regional performance, and top-selling products.

## Tools Used
- Python
- Pandas
- Jupyter Notebook
- Power BI

## Key Insights
- Total Sales: 2.29 Million
- Total Profit: 284.15 Thousand
- Total Orders: 5003
- Technology category generated the highest sales.
- West region recorded the highest sales performance.
- Top-selling products were identified through sales analysis.
